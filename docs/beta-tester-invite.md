## How the Course Works

1. Download the course materials (one zip file from GitHub)
2. Open Claude Code in the folder (type `claude` in your terminal)
3. Type `/start-0-0`
4. Claude guides you through 10 modules — step by step, no guessing

The whole thing runs inside the AI tool itself. Claude teaches you how to use it by using it with you.

## Questions

**Do I need coding experience?**
No. If you can open a terminal and type a command, you're set. We walk you through the setup.

**What is Claude Code?**
It's a terminal-based AI tool from Anthropic. It costs $20/month (Claude Pro or Max). It's different from the ChatGPT-style chatbot — it reads and writes files on your computer, runs multiple AI agents at once, and remembers your project across sessions.

**How long does it take?**
4–6 hours total. You can do it in one session or spread across a few days.

**What if I get stuck?**
Message Nick directly. During the beta, we're watching closely and will respond fast.

**Is this affiliated with Anthropic?**
No. This is built by ILT Academy using Claude Code as the delivery mechanism.
