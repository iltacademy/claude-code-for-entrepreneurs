# Claude Code for Entrepreneurs

**Free course by ILT Academy** — Learn to use Claude Code as a thinking partner, co-author, and research tool for your startup.

This course is built inside Claude Code itself. The tool is the classroom.

---

## What You'll Build

By the end of this course, you'll have produced real deliverables for your real business:

- **FOUNDER.md** — A permanent business context file Claude reads every session
- **Customer Insights doc** — Synthesized findings from your discovery interviews
- **Lean Canvas** — A complete, reviewed, presentation-ready one-pager
- **90-Day Proforma** — Your first financial model, with assumptions you can defend
- **5-Minute Pitch** — A Founder Showcase-ready pitch, refined by three AI agents

Not practice files. Your actual business.

---

## Prerequisites

- Claude Pro or Max subscription ($20/month at claude.ai)
- Mac, Windows, or Linux computer
- Basic familiarity with entrepreneurship concepts (ILT Academy Sprint 1 participants are ideal)
- No coding experience required

---

## How to Start

```bash
# 1. Install Claude Code
npm install -g @anthropic-ai/claude-code

# 2. Clone this repo
git clone https://github.com/iltacademy/claude-code-for-entrepreneurs.git
cd claude-code-for-entrepreneurs/course-materials

# 3. Open Claude Code
claude

# 4. Start the course
/start-0-0
```

## Course Structure

| Module | Lesson | What You Do |
|---|---|---|
| 0.0 | Setup | Build your FOUNDER.md — your permanent business context |
| 1.1 | First Session | Experience Claude working on your real business |
| 1.2 | Visualizing | Set up VS Code or Obsidian as a visual workspace |
| 1.3 | Reading the Room | Synthesize customer discovery interviews into insights |
| 1.4 | Three Voices | Run your lean canvas through 3 AI agents simultaneously |
| 1.5 | Building Your Team | Create custom agents tuned to your specific business |
| 1.6 | Permanent Memory | Lock in your business context so Claude always knows where you are |
| 2.1 | The Lean Canvas | Build your complete lean canvas, section by section |
| 2.2 | The Numbers | Build your 90-day financial model with AI as co-pilot |
| 2.3 | The Pitch | Draft and refine your 5-minute Founder Showcase pitch |

**Estimated time:** 4-6 hours total. Complete in one session or across multiple.

---

## The Three Agents

This course includes three built-in AI agents that review your work from different angles:

- **(💰) Investor** — Stress-tests your business model. Asks what a real investor asks.
- **(🧑) Customer** — Speaks from the perspective of your target user.
- **(⚔️) Competitor** — Maps the competitive landscape. Finds your gaps.

You'll also build a custom agent tuned to your specific business in Module 1.5.
